# Practica Qt Mini Office

Mini Word desarrollado en Python con PyQt6.

## Funcionalidades

- Ventana principal con título **"Mini Word"**.
- Área de texto principal (`QTextEdit`) para escribir y editar notas.
- Menú **Archivo**: Nuevo, Abrir, Guardar, Salir.
- Menú **Editar**: Deshacer, Rehacer, Cortar, Copiar, Pegar.
- Barra de herramientas con botones para las acciones principales.
- Panel lateral de **Buscar/Reemplazar** a la derecha:
  - Buscar siguiente / anterior / todas.
  - Reemplazar una / todas.
- Barra de estado:
  - Mensajes informativos.
  - **Contador de palabras**.
  - **Línea y columna del cursor** (funcionalidad extra).
- Personalización:
  - Cambio de color de fondo del área de texto.
  - Cambio de tipo de letra.

## Ejecución

```bash
python Unidad2/QT6_9MiniOffiPraFi.py
```
